Using Stripe

Stripe is a service which handles almost all of the details of collecting credit card payments from your customers. The basic flow is simple:

  1. Customers enter their credit card information in a form supplied by stripe.js, included within a form in our client HTML
  2. Submitting the form sends the credit card information to Stripe, which then returns to our client a "Stripe token", added as a hidden input "stripeToken" within our form
  3. On our server side, we then receive any other form input (e.g., name and address), along with the stripeToken
  4. On our server side, we use the Stripe client library (available for Ruby, Python, PHP and Java) to charge our customer using the stripeToken
