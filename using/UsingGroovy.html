Using Groovy

Groovy is a Java-based scripting language which runs on the JVM and offers dynamic typing and functional abstraction.

Groovy is freely available at http://groovy.codehaus.org. On Windows, simply download the latest binaries and unzip into your directory of choice. On Linux or Mac OS X, you may choose to simply download and untar, or else use a package manager such as apt-get or brew. After you've installed Groovy, check that it's working:

> groovy -v
Groovy Version: 2.2.1 JVM: 1.7.0_45 Vendor: Oracle Corporation OS: Mac OS X

We can experiment with Groovy code directly at the command line using the Groovy console:

> groovysh
Groovy Shell (2.2.1, JVM: 1.7.0_45)
Type 'help' or '\h' for help.
------------------------------------------------------------------------------------------------
groovy:000> 1 + 2
===> 3
groovy:000> quit

More typically, we'll write Groovy code as part of a project, using an IDE such as IntelliJ.
